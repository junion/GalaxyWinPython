Galaxy Communicator Documentation:

MITRE Utilities

MITRE is providing a library which exploits the power of MIT's Galaxy Communicator implementation in a number of ways, including:

  • Convenient tools for specifying frames
  • Tools for exploiting MIT's binary data type
  • Tools for brokering
  • Tools for adding a poll on stdin to the server loop
These tools are built on top of the Galaxy Communicator libraries, and do not modify them in any way.

See the page on compiling and accessing these utilities.
