MITRE Log Reference Documents

MITRE is providing a number of reference documents for use with its log tools. Some of these log tools implement standards which are being used in the evaluation process in the DARPA Communicator program. The most current version of these reference documents can always be found here.

  • MITRE logfile standard (version 14) (cf. the xmlize utility)
  • Logfile implementation of DMA metrics (version 4) (cf. the xml_annotate utility)
  • Sample human annotations file (cf. the xml_human_stubber utility)
  • Sample automatic annotation rules file (cf. the xml_annotate utility)
  • XML DTD for MITRE logfile standard (version 3) (cf. the xmlize utility)
  • XML DTD for human annotations file (version 2) (cf. the xml_human_stubber utility)
Note that there is no DTD for the automatic annotation rules files, because the rules framework for automatic annotation allows arbitrary attributes in the new: and not: namespaces, which cannot be validated against a DTD
